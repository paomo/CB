凯利公式 · 计算过程

1计算各情景收益率

收益率 = (到期回收金额 - 买入成本) / 买入成本。这是最基础的投入产出比——你花87.34元买入一张转债，最终可能收回多少钱，算出百分比盈亏。

关键点：回收价不是拍脑袋，每个情景背后有具体的逻辑支撑 ↓

买入价 P = 87.338元 情景A：安世部分恢复控制权 + 公司下修转股价促转股 → 到期正常赎回108元 rA = (108 - 87.338) / 87.338 = +0.2366 (+23.7%) 情景B：下修成功但转股不完全，剩余部分按面值兑付 → 回收约100元 rB = (100 - 87.338) / 87.338 = +0.1449 (+14.5%) 情景C：安世彻底脱表，公司仅剩境内业务，进入重整 → 重整方案回收约75元 rC = ( 75 - 87.338) / 87.338 = -0.1413 (-14.1%) 情景D：正股跌破1元面值退市，转债随正股退至老三板 → 流动性折价，回收约50元 rD = ( 50 - 87.338) / 87.338 = -0.4276 (-42.8%) 情景E：公司全面崩盘进入破产清算，资产打折处置 → 回收约20元 rE = ( 20 - 87.338) / 87.338 = -0.7710 (-77.1%)

2计算期望收益率 E[R]

期望收益率 = 每种情景的概率 × 对应收益率，全部加起来。这就是「平均来看，买这张转债赚还是亏」。

类比：掷骰子，1-2赢3倍，3-4不赢不亏，5-6输光。期望值 = 2/6×3 + 2/6×0 + 2/6×(-1) = 0.67，正数说明长期玩下去赚钱。

但这里算出来是 -6.23%——平均每投1元亏6分钱。

E[R] = Σ pi · ri （概率加权平均） = 0.30 × (+0.2366) → +0.0710 ← 30%概率赚23.7%，贡献+7.1% + 0.25 × (+0.1449) → +0.0362 ← 25%概率赚14.5%，贡献+3.6% + 0.20 × (-0.1413) → -0.0283 ← 20%概率亏14.1%，拖累-2.8% + 0.15 × (-0.4276) → -0.0641 ← 15%概率亏42.8%，拖累-6.4% + 0.10 × (-0.7710) → -0.0771 ← 10%概率亏77.1%，拖累-7.7% E[R] = -0.0623 = -6.23% 解读：正面情景（A+B）贡献 +10.72%，负面情景（C+D+E）拖累 -16.95% 负面拖累 > 正面贡献，所以期望值为负 期望回收 = 87.338 × (1 - 0.0623) = 81.9元/张 即：花87.34买入，概率加权平均只能拿回81.9元

3凯利公式：该投多少仓位？

凯利公式解决一个问题：如果一个投资机会你可以反复做，每次该下注多少比例的资金，才能让长期财富增长最快？

公式核心思想：几何增长率。假设你每次拿总资金的 f 比例来下注，反复多次后你的资金变成：

最终资金 = 初始资金 × (1+f·r₁) × (1+f·r₂) × (1+f·r₃) × ...

每一轮乘一个 (1+f·r)，r可能正可能负。要让乘积最大，取对数变成加法：

g(f) = Σ pi · ln(1 + f · ri)

g(f) 就是几何增长率，越大越好。找 f* 使 g(f) 最大，就是最优仓位。

关键性质：g(f) 在 f=0 处的斜率恰好等于 E[R]。如果 E[R]<0，斜率为负，意味着从0开始加仓只会让 g 下降——负期望的投资，任何仓位都不如空仓。

连续凯利公式： g(f) = Σ pi · ln(1 + f · ri) 代入本例 5 个情景： g(f) = 0.30·ln(1 + f·0.2366) + 0.25·ln(1 + f·0.1449) + 0.20·ln(1 + f·(-0.1413)) + 0.15·ln(1 + f·(-0.4276)) + 0.10·ln(1 + f·(-0.7710)) 对 f 求导： g'(f) = 0.30 × 0.2366/(1+f·0.2366) + 0.25 × 0.1449/(1+f·0.1449) + 0.20 × (-0.1413)/(1+f·(-0.1413)) + 0.15 × (-0.4276)/(1+f·(-0.4276)) + 0.10 × (-0.7710)/(1+f·(-0.7710)) 令 g'(f) = 0 求最优 f* 但在 f=0 处： g'(0) = 0.30×0.2366 + 0.25×0.1449 + 0.20×(-0.1413) + 0.15×(-0.4276) + 0.10×(-0.7710) = E[R] = -0.0623 斜率 < 0 → g(f) 在 f=0 处就开始下降 → f=0 就是最大值 f* = 0%，任何正仓位都降低长期增长率 直觉理解：你有一个亏钱的游戏，玩得越多亏得越多，最优策略就是不玩。 反过来，如果 E[R]>0（比如安世恢复概率升到50%以上），斜率变正，加仓能提升 g，这时候才有正的 f*。

4验证：不同仓位的实际增长率

下表把公式算一遍：假设你拿总资金的 f% 来买闻泰转债，经过一轮（到期兑付或违约），几何增长率 g 是多少？

所有 g 都是负数，且 f 越大越负 → 印证 f*=0 的结论。

注意 f=65% 时 1+f·rE = 1+0.65×(-0.771) < 0 → 意味着极端情景下本金归零甚至倒欠，数学上 ln 为负无穷，这就是过度下注的破产风险。
